SEMANA 3

🤖Construcción y maquetación del robot.

Diseño y recorte de materiales para la construcción del prototipo.

              “WALLY EN EL UNIVERSO”

Para terminar el carrito robot debemos unir cada una de las piezas, para su entrega final. 

1. Dibujar los moldes y recortar para darle forma a WALLY.

A continuación los moldes.  

2. Unir la parte de los circuitos y componentes electrónicos. 

El primer paso en la construcción de “WALLY EN EL UNIVERSO” consiste en ensamblar los componentes de hardware. como la base o chasis, ruedas, pilas recargables, interruptor y motores DC. 

Segundo paso es ubicar el Arduino uno, las salidas del puente H y se integran el módulo Bluetooth y los parlantes a la placa Arduino. 

También se debe ubicar un power wam que alimentara de energía para que se pueda mover, cada componente se coloca cuidadosamente, asegurando conexiones firmes y organizadas para evitar fallos durante el funcionamiento. Este ensamblaje es clave para garantizar que el robot cuenta con una estructura estable y funcional desde el inicio.
